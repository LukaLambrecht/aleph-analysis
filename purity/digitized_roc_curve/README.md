# Custom ROC curve plotter based on manually digitized figure

Use case: the paper "Measurement of AbFB using Inclusive b-hadron Decays" (https://arxiv.org/pdf/hep-ex/0107033)
contains a score distribution but no hard efficiency / mistag numbers to compare to.
The score distribution figure has been digitized using an online tool and the numbers stored in a csv file.
This script recreates the plot and also converts it into a ROC curve.
This is of course approximate, but good enough for internal reassurance.
